# Mine vs Rock Prediction

## 📌 Overview

This project builds a **Machine Learning classification model** that predicts whether an object detected by sonar signals is a **mine** or a **rock**. The model is trained using the **Sonar dataset** containing 60 numerical features representing sonar signal frequencies.

## ⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 🧠 Model

The project uses **Logistic Regression** for binary classification.

## 📂 Project Structure

```
Mine_vs_Rock_Prediction
│
├── Mine_vs_Rock_Prediction.ipynb
├── sonar_data.csv
└── README.md
```

## 🔍 Workflow

1. Load the dataset
2. Data preprocessing
3. Train-test split
4. Model training using Logistic Regression
5. Model evaluation using accuracy score

## 🎯 Goal

To build a model that can accurately classify sonar signals as **Mine (M)** or **Rock (R)**.
