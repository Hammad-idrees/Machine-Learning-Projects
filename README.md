# Machine Learning Projects

This repository contains a collection of **Machine Learning projects implemented using Python**. The goal of this repository is to practice and demonstrate different **machine learning algorithms, data preprocessing techniques, and model evaluation methods**.

More projects will be added to this repository as I continue learning and building new machine learning models.

---

## 📌 Technologies Used

The following technologies and libraries are used in these projects:

- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib  
- Seaborn  
- Jupyter Notebook

## 📂 Projects

### 1️⃣ Mine vs Rock Prediction

#### 📖 Project Description
A **classification model** that predicts whether an object detected by **sonar signals** is a **mine** or a **rock** using machine learning.

#### 🤖 Algorithm Used
- Logistic Regression

#### 📊 Dataset
- Sonar Dataset
